How it works

This calculator finds the exact n-th prime number for any 1 ≤ n ≤ 73,302,249,005 — covering all primes up to 2 trillion.

The algorithm runs entirely in your browser using WebAssembly:

  • Precomputed table — a 521 KB file (pi.dat) stores cumulative prime counts π(k) at every 30-million interval up to 2×10¹².
  • Binary search — given n, a binary search over the 66,667 table entries locates which 30M-wide block contains the n-th prime.
  • Segmented sieve — a sieve of Eratosthenes runs within that single block to find the exact prime. The small primes (up to √2×10¹² ≈ 1.41M) are precomputed once at startup.

Typical query time is under 1 second on modern hardware. No server involved — everything runs client-side.

pi.dat format

Binary file with an 8-byte magic (PIFIN01\0), followed by max_x, stride, records (each u64), 32 bytes padding, then records × u64 cumulative prime counts.
